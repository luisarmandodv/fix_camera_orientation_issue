# fix_camera_orientation_issue

## React Native Barcode Detection and Face Recognition Example

An example project demonstrating the use of [react-native-camera](https://github.com/react-native-community/react-native-camera) package and remplace the old RCTCamera component to the new RNCamera Component

See Old issue:
https://github.com/react-native-community/react-native-camera/issues/813#issuecomment-370944142

This demo was developed using this example: https://github.com/react-native-community/rncamera-example

### Features

Shows all different attributes of the camera like front and back cameras, flash modes, WB, etc.
Shows the usage of the Bar Code recognition.
Demonstrates the usage of the face detection feature by drawing polygons around the detected face.
